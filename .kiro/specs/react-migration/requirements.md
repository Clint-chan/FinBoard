# React + TypeScript 完整迁移需求

## 目标
将现有的原生 JavaScript 股票看板应用完整迁移到 React + TypeScript 技术栈，保留所有功能并提升代码质量。

## 功能清单

### 核心功能
1. **实时行情** - 腾讯 API 获取股票数据
2. **自动刷新** - 可配置的刷新间隔
3. **股票搜索** - 新浪 API 搜索添加股票
4. **价格预警** - 多条件预警设置
5. **持仓成本** - 成本价和盈亏计算
6. **主题切换** - 深色/浅色/自动主题

### 图表功能
7. **分时图** - 悬浮预览分时走势
8. **SuperChart** - 完整 K 线图组件
   - 分时图 / 5分 / 15分 / 30分 / 60分
   - 日K / 周K / 月K
   - MACD / RSI / VOL 副图指标
   - BOLL 布林带叠加
   - 十字光标交互
   - 滚轮缩放

### 交互功能
9. **拖拽排序** - 股票列表拖拽重排
10. **右键菜单** - 设置预警/成本/删除
11. **快捷键** - R刷新 / Esc老板键

### 高级功能
12. **分析大屏 (AnalysisDrawer)**
    - 左侧自选股列表
    - 中间 SuperChart 图表
    - 右侧 AI 聊天面板
    - 可拖拽调整宽度
13. **侧边栏** - 完整版导航和用户菜单
14. **云同步** - 登录/注册/配置同步
15. **老板键** - Esc 隐藏界面
16. **PWA** - Service Worker 离线缓存

## 技术要求
- React 18 + TypeScript 5
- Vite 5 构建
- 纯 CSS（无 UI 框架）
- 类型安全的 API 调用
- 组件化架构
- 自定义 Hooks 复用逻辑

## 部署目标
- Cloudflare Pages（静态前端）
- 保留现有 Cloudflare Workers API
